# 松戸ボードゲームサークル Booo! GAMES ウェブサイト

松戸ボードゲームサークル「Booo! GAMES」の公式ウェブサイトです。

## 概要

このウェブサイトは、松戸周辺で活動するボードゲームサークルの情報を提供し、初心者からゲーマーまで幅広い方々にボードゲームの楽しさを伝えるために作成されました。

## 主な機能

### 🎨 動的なインタラクション
- **スムーズなスクロールアニメーション**: セクションごとにフェードインアニメーション
- **カウントアップ統計表示**: 開催回数や参加者数がアニメーションでカウントアップ
- **今後の開催予定表示**: TwiPlaのイベント情報を動的に表示（モックデータ）
- **全FAQ常時表示**: よくある質問と回答を全て表示（アコーディオンなし）
- **レスポンシブナビゲーション**: モバイルとデスクトップに最適化されたメニュー
- **スクロール位置に応じたナビゲーションハイライト**: 現在表示中のセクションを強調表示
- **トップに戻るボタン**: スクロール時に表示される便利なボタン

### 📱 レスポンシブデザイン
- モバイル、タブレット、デスクトップに完全対応
- Tailwind CSSを使用した最新のデザイン
- カスタムアニメーションとトランジション効果

### 🎯 ユーザー体験の最適化
- Intersection Observerを使用したパフォーマンス最適化
- スムーズなページ遷移
- アクセシビリティに配慮した設計
- プリファードモーション設定への対応（`prefers-reduced-motion`）

### 🎮 リッチなコンテンツセクション
- **統計セクション**: 開催回数、参加者数、保有ゲーム数のカウントアップアニメーション
- **今後の開催予定**: イベントカードで参加者数と進捗バーを表示
- **人気ゲームギャラリー**: 6つの人気ボードゲームを紹介（カタン、ドミニオン、アグリコラなど）
- **参加者の声**: 6名の参加者レビューと評価を表示
- **拡張FAQ**: 8つのよくある質問を全て常時表示

### 🎲 楽しい隠し要素
- イースターエッグ機能（ロゴを5回クリック）
- 開発者コンソールメッセージ

## セットアップ方法

### 1. ロゴ画像の配置

`images/` フォルダに `logo.jpg` という名前でサークルのロゴ画像を配置してください。

画像がない場合、プレースホルダー画像が自動的に表示されます。

### 2. ウェブサイトの起動

このウェブサイトは静的なHTMLファイルで構成されているため、特別なセットアップは不要です。

#### ローカルで開く方法

**方法1: ブラウザで直接開く**
```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

**方法2: ローカルサーバーを使用（推奨）**

Pythonを使用する場合:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

その後、ブラウザで `http://localhost:8000` にアクセスします。

Node.jsの`http-server`を使用する場合:
```bash
# インストール
npm install -g http-server

# 起動
http-server

# または特定のポートで起動
http-server -p 8080
```

その後、ブラウザで `http://localhost:8080` にアクセスします。

**方法3: VS Codeの拡張機能を使用**

VS Codeを使用している場合、「Live Server」拡張機能をインストールして、右クリックから「Open with Live Server」を選択します。

## 使用技術

- **HTML5**: セマンティックなマークアップ
- **CSS3**: カスタムアニメーションとスタイリング
- **Tailwind CSS**: ユーティリティファーストCSSフレームワーク
- **Vanilla JavaScript**: フレームワークを使用しない純粋なJavaScript
- **Lucide Icons**: 美しいオープンソースアイコンライブラリ
- **Google Fonts**: Noto Sans JP フォント

## 主な JavaScript 機能

### モバイルメニュー (`initMobileMenu`)
- ハンバーガーメニューのトグル機能
- メニュー外クリックで自動的に閉じる
- スムーズなアニメーション

### スクロールアニメーション (`initScrollAnimations`)
- Intersection Observerを使用した効率的な要素の監視
- ビューポートに入ったときにフェードインアニメーション
- セクションごとのスタッガーアニメーション（順次表示）

### カウントアップアニメーション (`initCounterAnimation`)
- 統計数値が0からターゲット値までアニメーション
- Intersection Observerで表示時に一度だけ実行
- スムーズな60fpsアニメーション

### TwiPlaイベント読み込み (`loadTwiPlaEvents`)
- 今後の開催予定を動的に表示
- イベントカードに参加者数と進捗バーを表示
- モックデータを使用（TwiPlaに公開APIがないため）
- ローディング状態の表示

### ゲームカードアニメーション (`initGameCards`)
- 人気ゲームカードのフェードインアニメーション
- スクロール時の順次表示

### 参加者の声アニメーション (`initTestimonials`)
- レビューカードのスケールアニメーション
- スタッガー効果で順番に表示

### FAQ表示 (`initFAQ`)
- 全てのFAQを常時表示（アコーディオンなし）
- スクロールアニメーションのみ

### アクティブナビゲーション (`initActiveNav`)
- スクロール位置に応じて現在のセクションをハイライト
- パフォーマンス最適化のためのスロットリング

### パララックス効果 (`initParallax`)
- ヒーローセクションの背景要素にパララックス効果
- スクロールに応じた視差効果

## カスタマイズ方法

### 色の変更

Tailwind CSSのクラスを使用しているため、色の変更は簡単です。

例: 赤色のアクセントカラーを青色に変更
```html
<!-- 変更前 -->
<span class="text-red-500">...</span>

<!-- 変更後 -->
<span class="text-blue-500">...</span>
```

### セクションの追加

新しいセクションを追加する場合:

1. HTMLに新しいセクションを追加
2. セクションに `id` 属性を設定
3. ナビゲーションメニューにリンクを追加
4. 必要に応じてCSSとJavaScriptを調整

### アニメーションの調整

アニメーションのタイミングや動作は [`css/styles.css`](css/styles.css) で調整できます。
